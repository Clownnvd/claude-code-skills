# Page-Specific Bonus Criteria

Extra criteria beyond the 10 core categories, specific to page type.

## Landing Page Bonus

### Hero Section
- Headline visible in < 1 second
- Subheadline expands on headline (not repeats it)
- Hero image/video shows product, not stock photo
- Primary CTA above fold, secondary CTA visually subordinate
- Badge/announcement bar for urgency ("New: Feature X just launched")
- Social proof indicator near hero (avatar stack, star rating, user count)

### Proof Hierarchy
- Logos section after hero (trust anchoring)
- Features with specific benefits (not just feature list)
- Testimonials with measurable outcomes ("saved 3 weeks", "$450 MRR")
- Pricing transparent and easy to compare
- FAQ addressing top objections
- Final CTA reiterating value proposition

### Conversion Signals
- Single product/action focus (not multi-product catalog)
- No outbound links that pull users away
- Sticky header with CTA on scroll
- Mobile-optimized hero (text-first, image below)

## Dashboard Page Bonus

### Information Architecture
- Navigation: max 7Â±2 top-level items
- Current location clearly indicated (active nav state)
- Breadcrumbs for nested pages
- Search accessible from everywhere

### Data Presentation
- Key metrics visible without scrolling
- Charts use appropriate type (bar for comparison, line for trend)
- Empty states are helpful (guide user to first action)
- Loading skeletons match final layout shape

### Actions
- Primary action (Create/Add) prominently placed
- Destructive actions require confirmation
- Bulk actions available for lists
- Undo available for reversible actions

## Auth Page Bonus

### Trust & Security
- Brand identity clearly visible (logo, name)
- SSL indicator or security messaging
- Social login options (GitHub, Google) prominently placed
- Password requirements shown BEFORE typing

### Form UX
- Labels above inputs (not placeholder-only)
- Show/hide password toggle
- Error messages specific and next to field
- Auto-focus on first input
- Single-column layout (no side-by-side fields)
- Return key submits form

### Flow
- Link between sign-in / sign-up / forgot password
- Success state clear after submission
- Redirect to intended destination after auth

## Billing Page Bonus

### Pricing Display
- 2-5 plans maximum
- Recommended plan visually highlighted
- Feature comparison with checkmarks (not paragraphs)
- Annual/monthly toggle with savings percentage
- Price in local currency if possible

### Payment Form
- Credit card form auto-formats input
- Payment badges (Visa, Mastercard, Stripe secure)
- Money-back guarantee messaging
- Receipt/confirmation after payment
- Clear upgrade/downgrade/cancel paths
