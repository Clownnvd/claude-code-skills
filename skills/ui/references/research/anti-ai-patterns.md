# Anti-AI Patterns

Avoid these tells that make UI look AI-generated.

## Layout Red Flags

| Pattern | Problem | Fix |
|---------|---------|-----|
| Perfect 3-column grid | Too symmetric, robotic | Vary column widths, use asymmetric layouts |
| Equal spacing everywhere | No visual hierarchy | Use intentional spacing scale (tight groups, generous sections) |
| Centered everything | Feels like a template | Mix alignments — left-aligned text with centered headings |
| Card grids with identical heights | No content personality | Allow natural height, vary card types |

## Typography Red Flags

| Pattern | Problem | Fix |
|---------|---------|-----|
| Single font weight | Flat hierarchy | Use 3+ weights (regular, medium, bold) |
| Generic font (Inter only) | Every AI site uses it | Pair with a display font for headings |
| Same size headings | No rhythm | Use clear type scale (1.25-1.333 ratio) |
| Perfectly balanced paragraphs | Feels lorem-ipsum | Use real copy with natural lengths |

## Color Red Flags

| Pattern | Problem | Fix |
|---------|---------|-----|
| Blue gradient everything | Default AI palette | Choose brand-specific colors |
| Pure black text (#000) | Harsh, unrefined | Use near-black (zinc-900, slate-900) |
| Rainbow feature icons | Random, no system | Stick to 1-2 accent colors |
| Low contrast pastels | Inaccessible, washed out | Check WCAG AA (4.5:1 text, 3:1 UI) |

## Content Red Flags

| Pattern | Problem | Fix |
|---------|---------|-----|
| "Revolutionize your workflow" | Generic AI copy | Use specific outcomes ("Save 40 hours/month") |
| Placeholder testimonials | No name, no company, no photo | Use real quotes with attribution |
| Feature list with no hierarchy | All equally important | Highlight 1-2 hero features, secondary list |
| Stock photo hero | Impersonal | Use product screenshots, illustrations, or abstract |

## The Fixes — What Real Products Do

1. **Asymmetry**: Hero with text-left, image-right (not centered everything)
2. **Personality**: One bold design choice (color, typography, or layout)
3. **Specificity**: Real numbers, real names, real screenshots
4. **Restraint**: 2-3 colors max, 1-2 fonts, consistent spacing
5. **Motion with purpose**: Animate to guide attention, not to decorate
6. **Negative space**: Let elements breathe — whitespace is a feature
7. **Progressive disclosure**: Don't show everything at once
