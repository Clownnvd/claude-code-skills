## Search Strategy

### For specific components:
```
"[component] UI component 2026"
"best [component] design" site:21st.dev OR site:ui.aceternity.com OR site:magicui.design
"[component] react tailwind" site:github.com
```

### For trending patterns:
```
"UI design trends 2026 components"
"most beautiful [type] website 2026"
"trending web components" site:dev.to OR site:medium.com
```

### For specific styles:
```
"[style] [component] examples" (e.g. "glassmorphism navbar")
"[style] UI trend 2026"
"[style] design inspiration"
```
